# Gerer Events — Master Strategy Deck (Product + Business)

## 1) Vision
Gerer Events is a premium event lifecycle platform for modern hosts and planners.

It combines:
- Beautiful invitation experiences
- Reliable RSVP and QR check-in operations
- Post-event media and insights

Core promise: **Beautiful to share, reliable on event day, insightful after event day.**

---

## 2) Why This Wins
Most event tools fail in one of three places:
1. Design feels generic
2. Event-day operations are stressful
3. No meaningful post-event value

Gerer Events solves all three with a design-first and execution-ready product model.

## 5) Product Scope (MVP)
## Host Setup
- Create event (title, date, venue, type, story)
- Apply premium template and styling

## Invite Layer
- Shareable event link
- Individual QR pass generation

## RSVP Layer (one-page)
- Name
- Contact
- Yes/No/Maybe
- Plus-one count
- Meal option (optional)
- Message to host (optional)

## Event-Day Layer
- Fast QR scanner mode
- Live attendance updates

## Post-Event Layer
- Upload access: checked-in guests only
- Default: auto-approve uploads
- Optional: review mode
- Host controls: feature/hide/reject/download album

## Dashboard Priority (locked order)
1. RSVP rate
2. Actual check-ins
3. Media uploads
4. Plus-ones
5. No-show rate
